Delivery Model

No container image. Just the right artifact for VM deployment.

01

Package

OCI DevOps build creates a binary archive from the static site, install script, and version file.

02

Deliver

The pipeline publishes the archive as a deployable artifact for the compute instance group stage.

03

Roll Out

The deployment spec runs through Compute Instance Run Command and updates private instances in batches.

Architecture

Built for private backends and public validation.

The instances stay private. The load balancer is the only public entry point, while OCI DevOps executes the install workflow through the instance agent.
